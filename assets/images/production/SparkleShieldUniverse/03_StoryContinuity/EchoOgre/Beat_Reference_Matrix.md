## Folder Use

- Put finished spread art in that beat's folder.
- Put prompt drafts or art notes in that beat's folder.
- Put revised images in the same beat folder with version numbers.
- Keep reusable character and environment references in `01_Characters` and `02_Environment`; do not duplicate them inside beat folders.
