# Sparkle Shield Universe Production Assets

Start here when you need references for a new illustration, spread, story page, prompt, or future book.

## Fastest Way To Find Assets

1. Open `00_Production_Guide/Quick_Find_Asset_Map.md`.
2. Pick the task: character, pose, expression, costume, location, lighting, environment state, or Echo Ogre beat.
3. Use the listed Production ID files in your image prompt.

## Canonical Spread Records

For book production decisions, use the story manifest before older planning notes:

- `03_StoryContinuity/EchoOgre/spreads.json` is the canonical spread record for *The Echo Ogre*.
- Beat folder `README.md` files are living workspaces for prompt notes, thumbnails, revisions, and team feedback.
- `Beat_Reference_Matrix.md` is a planning map. If it disagrees with the manifest, the manifest wins until the team explicitly changes it.

## Main Folders

- `00_Production_Guide/` - indexes, naming rules, and quick-find maps.
- `01_Characters/` - model, construction, action, expression, costume, and pose references.
- `02_Environment/` - world boards, Maplewood Terrace maps, location sheets, and environmental-state references.
- `03_StoryContinuity/` - story-specific beat planning folders and reference matrices.

## Production ID Rule

Use Production IDs first, descriptive names second.

Examples:

- `BBK` = Bubble Boy Kairo
- `CC` = Captain Cash
- `DAD` = Dad
- `MOM` = Mom
- `NN` = Ninja Nugs
- `EOG` = Echo Ogre
- `MT01` = Family Home
- `MT01I` = Kitchen
- `MT04` = Neighborhood Green
- `MT06` = Floating Garden
- `ENV` = World, emotional rules, continuity, lighting, or environment-state reference

This keeps the system searchable and expandable as more books, spreads, and characters are added.
